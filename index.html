COMO FUNCIONA?

O Code in the Dark funciona assim: Você recebe um layout e tem 15 minutos para reproduzi-lo com HTML e CSS, complicado né?

Mas temos mais alguns pequenos detalhes: Você não pode usar qualquer programa de edição e conta apenas com uma imagem do layout. Além disso, sem previews! Só vai poder ver como ficou no fim da rodada. Animal né?

São 04 rodadas com 04 competidores em cada, o vencedor de cada rodada disputa a grande final!

INSCRIÇÕES

O evento é 100% grátis, mas temos limitações de espaço. As vagas serão divididas da seguinte forma: 90% das vagas serão sorteadas entre os 100 primeiros inscritos do Front In Vale e os outros 10% entre o restante dos inscritos.

Mesmo os inscritos no Front in Vale precisam preencher o formulário do Code in the Dark.

Para saber se foi selecionado ou não, fique atento que receberá um e-mail na semana do evento.
